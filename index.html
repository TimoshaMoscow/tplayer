https://timoshamoscow.github.io/tplayer/player.js

📌 TPlayer распространяется только через CDN — это обеспечивает автоматическое обновление и лучшую производительность.

Быстрый старт

1

Подключите CDN

Добавьте ссылки в ваш HTML

<link href="player.css">
<script src="player.js">
2

Добавьте контейнер

Создайте элемент для плеера

<div id="player"></div>
3

Инициализируйте

Укажите источник видео

new TPlayer('#player', {
  source: 'video.mp4'
});
const player = new TPlayer('#player-container', {
    source: 'test.mp4',      // ваше видео
    poster: 'poster.jpg',    // постер (опционально)
    accentColor: '#ff3b3b'    // акцентный цвет
});

💡 Всё работает из коробки! Плеер сам определит качество видео и создаст меню выбора. Опция "Оригинал" позволяет вернуться к исходному качеству в любой момент.
