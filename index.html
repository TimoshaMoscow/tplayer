Установка

<link rel="stylesheet" href="https://timoshamoscow.github.io/tplayer/player.css">
<script src="https://timoshamoscow.github.io/tplayer/player.js"></script>

<div id="player"></div>

<script>
    const player = new TPlayer('#player', {
        source: 'test.mp4',
        poster: 'poster.jpg',
        accentColor: '#ff3b3b'
    });
</script>
player.css player.js
1

Подключите CDN

<link href="player.css">
<script src="player.js">
2

Добавьте контейнер

<div id="player"></div>
3

Инициализируйте

new TPlayer('#player', {
source: 'test.mp4',
poster: 'poster.jpg'
});

💡 Простота использования: Вам не нужно указывать разные качества видео. Просто укажите source: "test.mp4" и плеер сам определит качество! Опция "Оригинал" в меню качества позволяет вернуться к исходному видео.
