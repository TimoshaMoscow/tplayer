Установка

Подключите TPlayer через CDN — это единственный способ установки библиотеки

🌐 CDN ссылки

https://timoshamoscow.github.io/tplayer/player.css
https://timoshamoscow.github.io/tplayer/player.js 
<!-- Подключение TPlayer (единственный способ установки) -->
<link rel="stylesheet" href="https://timoshamoscow.github.io/tplayer/player.css">
<script src="https://timoshamoscow.github.io/tplayer/player.js"></script>
📌 Важно: TPlayer распространяется исключительно через CDN. Это обеспечивает автоматическое обновление и лучшую производительность.

Быстрый старт

✅ Проще некуда! Вам не нужно указывать разные качества видео. Достаточно одного источника.

1

Подключите CDN

Добавьте CSS и JS ссылки в ваш HTML-документ

<link rel="stylesheet" href="https://timoshamoscow.github.io/tplayer/player.css">
<script src="https://timoshamoscow.github.io/tplayer/player.js"></script>
2

Добавьте контейнер

Создайте DOM-элемент для плеера

<div id="my-player"></div>
3

Инициализируйте плеер

Всего одна строка — укажите путь к видео

const player = new TPlayer('#my-player', {
    source: 'video.mp4',     // ← просто укажите ваше видео
    poster: 'poster.jpg',    // постер (опционально)
    accentColor: '#ff0000'    // акцентный цвет (опционально)
});
💡 Вот и всё! Плеер сам определит качество вашего видео и создаст меню выбора качества. При выборе более низкого качества видео автоматически сжимается на лету. Опция "Оригинал" позволяет вернуться к исходному качеству в любой момент.
📹 Пример с реальным видео: 
// Если у вас есть файл my-video.mp4
const player = new TPlayer('#player', {
    source: 'my-video.mp4',
    poster: 'preview.jpg'
});

🎯 Плеер автоматически:
• Определит разрешение видео (360p, 720p, 1080p и т.д.)
• Создаст меню со всеми доступными качествами до вашего
• Добавит опцию "Оригинал" для возврата к исходному качеству
• При выборе низкого качества сожмёт видео без потери времени
