Установка

<link rel="stylesheet" href="https://timoshamoscow.github.io/tplayer/player.css">
<script src="https://timoshamoscow.github.io/tplayer/player.js"></script>

<div id="player"></div>

<script>
    const player = new TPlayer('#player', {
        source: 'test.mp4',
        poster: 'poster.jpg',
        accentColor: '#ff3b3b'
    });
</script>

Быстрый старт

1

Подключите CDN

<link href="player.css">
<script src="player.js">
2

Добавьте контейнер

<div id="player"></div>
3

Инициализируйте

new TPlayer('#player', {
source: 'test.mp4'
});

💡 Всё просто! Укажите source — плеер сам определит качество видео. В меню качества доступна опция "Оригинал" для возврата к исходному видео.
