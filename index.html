Быстрый старт

1

Скачайте библиотеку

Нажмите на кнопку выше и скачайте файлы player.js и player.css

📁 Ваш проект/
  ├── player.js
  ├── player.css
  └── index.html
2

Подключите файлы

Добавьте стили и скрипт в ваш HTML-документ

<link rel="stylesheet" href="player.css">
<script src="player.js"></script>
3

Добавьте контейнер

Создайте DOM-элемент для плеера

<div id="my-player"></div>
4

Инициализируйте плеер

Создайте экземпляр TPlayer с вашими настройками

const player = new TPlayer('#my-player', {
    sources: {
        '360p': 'video-360.mp4',
        '720p': 'video-720.mp4',
        '1080p': 'video-1080.mp4'
    },
    poster: 'poster.jpg',
    accentColor: '#ff0000'
});
💡 Совет: Убедитесь, что ваши видеофайлы доступны по указанным путям. Плеер автоматически выберет лучшее доступное качество.
